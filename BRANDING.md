# Vertex 360° — Guia de Branding

## Identidade

### Nome oficial
**Vertex 360°**

### Slogan principal
> Inteligência Estratégica para Mandatos

### Tagline
> Decisão baseada em dados, não em achismo.

### Descrição curta
Vertex 360° é uma plataforma que integra dados territoriais, gestão de mandato e suporte à decisão estratégica em um único lugar.

### Descrição longa
Vertex 360° integra dados territoriais, CRM político, inteligência territorial e suporte à decisão de forma contínua — desde a organização da base até a execução do mandato com resultados mensuráveis.

## Sobre a Mudança de Nome

O projeto foi originalmente concebido como **"Meu Mandato 360°"**. Em 2025, evoluiu para **Vertex 360°**, refletindo um reposicionamento estratégico:

- De "gestão de mandato" para **"inteligência estratégica"**
- De ferramenta operacional para **plataforma de suporte à decisão**
- De foco administrativo para **integração territorial + dados + estratégia**

O nome "Vertex" (vértice) simboliza o ponto de convergência onde dados, território e decisão se encontram.
