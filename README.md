# Vertex 360° — Inteligência Estratégica para Mandatos

> Decisão baseada em dados, não em achismo.

## Sobre o Projeto

**Vertex 360°** é uma plataforma de inteligência estratégica para mandatos políticos. Integra dados territoriais, CRM político, inteligência territorial e suporte à decisão de forma contínua — desde a organização da base até a execução do mandato com resultados mensuráveis.

### Visão

Ser a plataforma de referência para gestores públicos que governam com método, dados e transparência.

### Objetivo

Centralizar dados territoriais, projetos, base política e indicadores em um único lugar, transformando informação dispersa em decisões estratégicas.

## Sobre a Mudança de Nome

Este projeto foi originalmente desenvolvido sob o nome **"Meu Mandato 360°"**. Em 2025, o produto evoluiu para **Vertex 360°**, refletindo um reposicionamento de "gestão de mandato" para **"inteligência estratégica para mandatos"** — com foco em dados territoriais, suporte à decisão e resultados mensuráveis.
